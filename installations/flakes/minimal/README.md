# Minimal Finix

This is my minimal configuration for finix. It installs absolutely nothing but
the things needed to work on and expand a finix configuration.

I am not promising you a perfect system that will work exactly how you want with
services you want. I am promising you a system that boots and is self sufficient
enough to get it where you want eventually.

# Requirements

- Nix Flakes
- Internet
